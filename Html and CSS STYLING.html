CSS Flexbox: flex-wrap[1]

With a flexbox layout, by default flex items will all try to fit onto one line.You can change that and allow the items to wrap as needed with this property.

Usage

This property supports the following values:

  • nowarap(default): all flex items will be on one line.
  • wrap: flex items will wrap onto multiple lines, from top to buttom.
  • wrap-reverse: flex items will wrap onto multiple lines from bottom to top.

Example

            
 .container{
    flex-wrap: nowrp | wrap | wrap-reverse;
 }
